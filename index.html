Fundamental aspects fundamental

The WRIO Internet OS platform greatly improves the browsing experience of the everyday Internet user. By utilizing blockchain data storage and it’s ability to analyze and output machine-readable data, the user experience is exponentially improved compared to the current norm.

Machine-readable

The format of web 3.0 pages is machine-readable, which enables automatic data processing.

Automatic Data Processing · ADP

ADP paves the way to the development of automatic curated web: metadata of each created page is recorded in the blockchain, then it is correlated to the user’s interests stored in the profile. If there is a match, the page is displayed in the user’s Web Dashboard.

Web Dashboard

Automatic arrangement of bookmarks based on the context: news, games, music, social media, and etc. Web Dashboard is a key element of user-centric web and web activity automation.

Cloud & Dapp Web 3.0 Platform

Rich functionality through cloud/distributed apps: no need to download, update, install or setup any software.

Secure And Private

User-controlled disclosure, malware and viruses free, efficient protection from spam.

Minimum viable product

The platform will upgrade today’s Web 2.0 to machine-readable Web 3.0. It will pave the way for automatic data processing and will enable user-centric features: users will receive screened information automatically based on their interests

Thin-client Terminal Devices

Automatic correlation, classification, and cataloging of web data against users’ interest matrices will allow the creation of predictive user-centric services and next generation search engines. WRIO Internet OS will become their platform, and will create a new market of browser-driven smart predictive systems — very inexpensive thin-client terminal devices with WRIO Internet OS onboard
